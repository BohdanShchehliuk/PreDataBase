Сайт для пошуку деталей до машин

Кожен клієнт під час реєстрації повинен ввести такі обовязкові дані (стрінг):
0. опис прохання клієнта: registration; data_renewal; password_renewal;get_information
1.логін
2.пароль // пароль має містити не менше 6 символів, (цифри, символи, латинські літери) [0-9+!@#$%^&*+A-Z]
3.імя
4.прізвище
5.рік народження


Кожен клієнт може внести такі  дані (стрінг):
6. марка машини
7. він код машини
8. кодове слово (для відновлення паролю).
9.новий пароль - завжди на позиції 9 (у разі потреби відновлення пароля)

Максимальна позиція у файлі (кількість рядків): 20  - якщо потрібно буде дописати іншу інформацію;


Нумерація відповідно до принципу використання індексу у масиві (0 індекс - це перший рядок).


Алгоритм роботи програми.
1. Клас Main - зчитує шлях файла і передає його у клас Read
2. Read зчитує інформацію у файлі і передає її на перевірку у клас Check
3. Клас Сheck відповідає за наступне:
-чи є правльно задане слово прохання клієнта (регістрація, отримання даних...)
-якщо так, то чи відповідає файл всім вимогам (наприклад, під час реєстрації, чи відповідає пароль вимогам, чи не є клієнт
уже створений і так далі)
4. Якщо все добре то клієнта записують у базу даних за допомогою класу Write (а також під час
реєстрації додають до масиву даних Clients static Map<String , List<String>> clients )

Регістрація - клас Registration
перед регістрацією відбуваєтсья перевірка бази Clients static Map<String , List<String>> clients.
якщо ваш логін уже існує -  вам відмовляється у реєстрації.
При регістрації дані вводяться з 0 по 8 рядок

Приклад запиту:
registration
bohdan1987
111111
Bohdan
Shchehliuk
1987
mazda
22222
mazda2008


Оновлення даних - клас Renewal.
Якщо ви ввели вірний пароль і логін, то програма перезаписує дані.
При оновленні даних інформація вводяться з 0 по 8 рядок

Приклад запиту:
data_renewal
bohdan1987
111111
Bohdan
Shchehliuk
1987
mazda3
3333
magicWord

Оновлення пароля - клас Password_renewal.
Користувач повинен вкзатати солово команду на 0 позиції, вказати інформацію, яку він памятає
з 1- по 8 позицію і на 9-ій позиції вказати новий пароль.
Якщо кількість балів, яку він набере в результаті перевірки >100, а також пароль
відповідає певним критеріям (таким самим, як під час реєстрації), то його дані перезапишуть у файл.

Приклад запиту:
password_renewal
bohdan1987
b             // все, що памятає користувач з пароля
Bohdan
Shchehliuk
1987
mazda
22222
magic_word
b1987№%22222

Отримання інформації - клас Get_information
Користувач повинен вказати вірно слово-команду, логін та пароль.
В результаті він отримає з бази даних інформацію про свої дані.

Приклад запиту:
get_information
bohdan1987
b1987№%22222
